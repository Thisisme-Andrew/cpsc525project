## How to Run the Application

1. Use [pip](https://pypi.org/project/pip/) to install the required python packages

    ```bash
    pip install -r requirements.txt
    ```

2. Navigate into the source directory:
    ```bash
    cd src
    ```

3. Run the Python application:
    ```
    python app.py
    ```
